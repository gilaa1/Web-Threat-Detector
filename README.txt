Web Threat Detector Chrome Extension

Overview:
The Web Threat Detector Extension aims to protect you from malicious sites!
To help us protect more users, feel free to report potential threats, using our popup,
a simple interface to quickly report any suspicious activity.

Installation:
- Clone or download the repository to your local machine.
- Make sure you add your Firebase api key
- Open Chrome and navigate to chrome://extensions/.
- Enable "Developer mode" by toggling the switch in the top right corner.
- Click on "Load unpacked" and select the directory where you downloaded the 
    extension. (this folder)


Functionality:
the extension will automatically warn you when entering a suspicious website.

Reporting Usage:
Click on the Web Threat Detector icon in the Chrome toolbar.
A popup will appear with the extension interface.
Click the "Report Threat" button to report any suspicious activity.
